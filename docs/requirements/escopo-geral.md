# Desafio Técnico – Microsserviços Bancários

## Contexto
Você fará parte de uma equipe responsável por desenvolver um sistema bancário inovador,
onde modularidade e comunicação eficiente entre microsserviços são cruciais.

O objetivo é criar **dois microsserviços independentes**:
- Microsserviço de Transações
- Microsserviço de Clientes

A comunicação entre eles deve ser **desacoplada e eficiente**.

---

## Organização e Autonomia
- Total liberdade para organizar tempo e estrutura
- Comunicação assíncrona via documentação
- Contato com o time seletivo apenas para dúvidas essenciais
